安装WSL并安装图形界面

参考

https://zhuanlan.zhihu.com/p/146146238

开始

打开控制面板——程序——启用或关闭Windows功能——适用于Linux的Windows子系统

安装Linux

在微软商店中搜索Linux，即可安装
这里选的是Ubuntu

更换源

sudo sed -i s@/archive.ubuntu.com/@/mirrors.aliyun.com/@g /etc/apt/sources.listsudo 
apt-get update && sudo apt-get upgrade -y

安装xfce4

sudo apt-get install xfce4 -y

添加环境变量

export DISPLAY=localhost:0 
source /etc/profile

打开XLaunch

访问图形界面
